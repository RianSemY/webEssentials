Como fazer senha em html

Para fazer uma senha você precisará de dois comandos básicos, são eles:

< label for="(Senha)">(nome para aparecer na caixa da senha)< /label> (sem os espaços)
< input type="(tipo de texto, nesse caso, passoword)" name="(a senha em si)">
É usado o type para ficar "..." e não mostrar a senha
